Luminous Code (OSS)#

Luminous Code is an open-source collection of projects, namespaces, classes, methods, and extention methods, that I wrote to keep the code that I need to write in my various projects as DRY as possible! It's driven by my own needs as a developer, but I'm more than happy to share it with anyone who might find it useful, and to accept suggestions or bug reports.

This is the second (and current) open-source iteration of the original closed-source Luminous Framework.

You can find the website here, and the github repo here.
