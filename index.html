Background#

Luminous Software Solutions is an independent, privately owned Australian entity, based in Melbourne, Australia.

In 2011, I

  • founded Luminous Software (now Luminous Software Solutions)
  • created the LightSwitch Central website (which no longer exists)
  • co-authored a book called Pro Visual Studio LightSwitch 2011 Development (for more information see Yann's Book )
  • created some of the first community-based LightSwitch extensions (for more information see Yann's Extensions)

Although at that time the extensions weren't open-source, they were released to the community for free.

Luminous Software Solutions develops:

  • custom desktop applications for clients
  • custom web applications for clients
  • proprietary Visual Studio Extensions for internal use
  • open-source Visual Studio extensions for the Visual Studio community

Yann Duran holds the position of Senior Software Developer at Luminous Software Solutions.
